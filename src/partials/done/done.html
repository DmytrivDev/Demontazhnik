Виконані роботи

  • Миколаївська область

    Січень-Червень 2024

    Демонтаж трьох шкіл, адмінбудівлі загальною площею 13500 м²

    Демонтаж розпочався вручну через обмежений доступ до ділянки. Металеві конструкції розрізали, будівельне сміття сортували та вивозили, звільняючи місце для спеціальної техніки та вцілілого устаткування, яке передали власнику.

    Потім було задіяно екскаватор для демонтажу та вивозу сміття, автокран і автовишку для демонтажу плит перекриття, ферм та металоконструкцій промислової
